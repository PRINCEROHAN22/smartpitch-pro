# SmartPitch Pro – AI Outreach Tool (CLI + Web UI) 🚀

💬 Instantly generate personalized cold DMs for freelancers, marketers, and founders.

⚙️ **No API Needed • Works Offline • Includes CLI + Flask Web UI**

---

## 🧰 Features

✅ SmartPitch CLI – Classic terminal-based outreach tool  
✅ SmartPitch UI – Clean Flask HTML interface (form input + response)  
✅ Personalization: your name, client name, platform, offer  
✅ Output: Cold DM pitch auto-generated using smart templates  
✅ Runs on Python 3.8+ – No external APIs or accounts required

How to Use:

Note: (use cd command change the directory and navigate accordingly, i will attach images to avoid confusions thanks you !!!) 

1. CLI:
   Run using:
   python smartpitch_cli.py
   Follow prompts to generate a pitch.

2. Web UI:
   Run using:
   python app.py
   Open 127.0.0.1:5000 in browser.
   Fill form → Get outreach message.

💡 Requirements:
- Python 3.8+
- Flask (install via pip if not available)
